QuickActions

40+ Sozial-Animationen — Hug, Kiss, Slap — direkt per Klick auf jeden Spieler.

Dot-Taste öffnet das Emote-Wheel. Sofort erreichbar, mitten im Spiel.

Recolorable GUI

Komplette Farbpalette anpassbar. Eigenes Design, eigener Stil.

Smartbar Tabs

Home · Character · Scripts · Actions · Playerlist · Settings — sauber oben.

Game Scripts

Eingebaute Hubs für MM2, Bladeball und 99 Nights. Direkt aus dem Scripts-Tab.
